## ğŸš€ Getting Started

Follow these steps to run the project locally.

### Prerequisites

Make sure you have the following installed:
- [Node.js](https://nodejs.org/)
- A package manager like `npm` or `yarn`.
- Modern browsers like **Google Chrome** or **Safari**.

### Installation

1. Clone this repository:
    ```bash
    git clone https://github.com/thilakshan2003/macbooksite
    ```

2. Navigate to the project directory:
    ```bash
    cd macbooksite
    ```

3. Install dependencies:
    ```bash
    npm install
    ```

4. Start the development server:
    ```bash
    npm start
    ```

5. Open your browser and go to:
    ```
    http://localhost:3000
    ```

### Build for Production

To build an optimized version of the project for deployment:
```bash
npm run build
```
The production files will be located in the `build/` folder.
